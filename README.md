# IT Help Desk Ticketing System

## Overview
A Python-based ticket management system simulating real-world IT support workflows.

## Features
- Create support tickets
- Assign priority levels
- View all tickets
- Close tickets
- Track ticket status

## Technologies Used
- Python

## Purpose
This project demonstrates understanding of IT workflow processes and basic system design.

## Future Improvements
- Add database storage
- Convert to web app
- Add user authentication
